A note about learning vim.

Chapter 1: The Vim Way

Tip 1: Meet the Dot Command

. is to repeat the last change

  • The Dot Command is a Micro Macro.

Tip 2: Don’t Repeat Yourself

Move and use the dot command

  • Reduce Extraneous Movement

Tip 3: Take One Step Back, Then Three Forward

  • Make the change repeatable
  • Make the motion repeatable
  • All together

Tip 4: Act, Repeat, Reverse

Notice that many commands have its reversed form to themselves.
So you can undo almost everything.

Tip 5: Find and Replace by Hand

  • Use :%s/content/copy/g to replace all in this file
    • Or use /content and . to replace one by one with your judgement.

Tip 6: Meet the Dot Formula

  • One Keystroke to Move, One Keystroke to Execute.

Chapter 2: Normal Mode

Tip 7: Pause with Your Brush Off the Page

When you ask yourself whether you should enter the normal mode, then do it.

Tip 8: Chunk Your Undos

  • u is the undo command.
    • u and <C-r> are relative commands.

Tip 9: Compose Repeatable Changes

  • In a word, make your command repeatable as possible as you can,
    so when you find the next operator can be done by ., you’ll be happy.

Tip 10: Use Counts to Do Simple Arithmetic

  • <C-a> and <C-x> perform addition and substraction on numbers.

VimSheet

  • This sheet is associated by myself which I think it is easy to remember.

  • . is to repeat the last change

Basic movement

  • ‘j’ is to move cursor down.
  • ‘k’ is to move cursor up.
  • ‘h’ is to move cursor left.
  • ‘l’ is to move cursor right.
  • ‘0’ is to move cursor to the beginning of the line.
  • ‘^’ is to move cursor to the first non-space character.
  • ‘$’ is to move cursor to the end of the line.

Enter insert mode

  • a is to enter the insert mode after the cursor.

  • A is to enter the insert mode at the end of the line.

  • i is to enter the insert mode before the cursor.

  • ‘I’ is to enter the insert mode at the beginning of the line.

  • ‘o’ is to enter the insert mode on the next line.

  • ‘O’ is to enter the insert mode on the previous line.

  • ‘s’ is to delete the current letter and enter the insert mode.

  • ‘S’ is to delete the entire line and enter the insert mode.

  • ‘c [command]’ is to delete something and enter the insert mode.

    • What to delete is defined by [command].

  • ‘C’ is to delete from the cursor to the end of line and enter the insert mode.

  • d/D is similar to ‘c/C’ but they don’t enter the insert mode after deleting.

Find something to move

  • f [l] is to find and move to the first matched letter afterward in this line.
  • F [l] is to find and move to the first matched letter forward in this line.
  • t [l] is to find and move before the first matched letter afterward in this line.
  • T [l] is to find and move before the first matched letter forward in this line.
    • ; can repeat the last f/F/t/T command in the same direction.
    • , can repeat the last f/F/t/T command in the opposite direction.
  • /pattern is to find “pattern” afterward in this file.
  • ?pattern is to find “pattern” forward in this file.
    • n can repeat the last / or ? command in the same direction.
    • N can repeat the last / or ? command in the same direction.
